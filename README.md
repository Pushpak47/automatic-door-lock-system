# Automatic Door Lock System

This project demonstrates an automatic door locking system using a microcontroller. The system improves security by allowing controlled access using electronic authentication.

## Project Overview
The automatic door lock system uses a microcontroller and sensors to control a door locking mechanism. When the correct input is provided, the system unlocks the door automatically.

## Features
- Automatic locking and unlocking
- Improved security
- Easy to use electronic access system

## Hardware Components
- Microcontroller (Arduino / ESP32)
- Servo motor / Solenoid lock
- Keypad / RFID / Password input
- Power supply
- Door lock mechanism

## Applications
- Home security
- Office access control
- Smart door systems
